## Method 2: Compress Your Files

Compression helps with documents and images, but barely touches videos, photos, MP3s, or PDFs — those are already compressed. Don't waste time zipping them.

**Windows:** Right-click → **Send to** → **Compressed (zipped) folder**
**Mac:** Right-click → **Compress "[filename]"**

For videos, compression actually works — but you need to re-encode, not zip. Use [Vert.sh](https://vert.sh/) or [HandBrake](https://handbrake.fr/) to drop resolution to 720p. See: [How to compress a video](/how-to/compress-a-video).

## Method 3: Split Large Files into Parts

If compression isn't enough, split your file into smaller chunks that fit Discord's limit.

### Using 7-Zip (Windows/Mac/Linux):

Install [7-Zip](https://www.7-zip.org/), right-click your file → **Add to archive** → set **Split to volumes** to `9M`. Upload each part to Discord. Recipients download all parts into the same folder and extract with 7-Zip.

<Warning>
**Important:** Recipients need ALL parts in the same folder to extract. Make sure they download every part before trying to extract.
</Warning>

**WinRAR alternative:** Right-click → **Add to archive** → set **Split to volumes** to `9M`.

---

## Method 4: Cloud Storage Link

If you already use Google Drive, Dropbox, or OneDrive, upload there, set the link to "anyone with the link," and paste it in Discord. Good if you want the file to stay around — bad if it eats your storage quota.

## FAQ

**Can I get banned for sharing external links on Discord?**
No. Discord explicitly allows external links for files over the limit.

**Can I send large files in Discord DMs?**
Same 10 MB limit applies. Use any of the methods above.

**Is Nitro worth it just for file uploads?**
If you regularly send 10-500 MB files and want the other perks, sure. For occasional larger files, a free transfer service is enough — and Nitro caps at 500 MB anyway, so anything bigger still needs a link.
